# AutoML Engine — AI アプリケーション向け機械学習基盤
[English README is here](README.md)

---

## 概要
本プロジェクトは、AI アプリケーションへの組み込み利用を前提とした Python 製 AutoML エンジンです。
分類・回帰・クラスタリング・異常検知に対応し、学習・評価・説明をプログラムから制御できます。
パッケージとして提供され、学習結果を LLM により解釈する機能を備えています。UI・UX は対象外です。

本ライブラリは、機械学習を業務システムに組み込むエンジニアを主な対象としています。

---

## 設計方針
- I/O や永続化方式を固定しない
- タスクごとに責務を分離する
- 上位レイヤーからの制御を前提とする

---

## 主な機能
分類・回帰・クラスタリング・異常検知に対応した AutoML を中核に、再現性のある学習制御からモデル評価・最適化、説明可能性、LLM による結果要約までを一貫して実行する機能を提供します。

- **AutoML 実行**  
  分類・回帰・クラスタリング・異常検知といった各タスクに対して、モデル学習から評価までの一連の処理を自動で実行します。

- **モデル学習・選択制御**  
  使用するアルゴリズムや前処理、乱数シードなどをプログラムから明示的に指定でき、実験条件の再現性を確保できます。

- **評価指標の算出とモデル比較**  
  タスクに応じた評価指標を算出し、複数モデルの性能を比較したうえで最良モデルの選択を支援します。

- **ハイパーパラメータ最適化**  
  Grid Search や Optuna などの探索手法を切り替えて利用し、効率的なハイパーパラメータ最適化を行えます。

- **説明可能性**  
  SHAP や LIME を用いて、モデル予測に寄与した特徴量やその影響度を算出します。

- **学習結果の構造化出力**  
  モデル本体、評価指標、特徴量重要度などの学習結果を、再利用しやすい構造化された形式で取得できます。

- **LLM による結果要約**  
  学習結果や評価内容を、LLM を用いて自然言語で要約します。

## 対象範囲
本プロジェクトは、AutoML エンジンとしての中核機能に責務を集中させる設計を採用しています。  そのため、以下の領域は設計上の前提として本エンジンの提供範囲には含めていません。

- **ユーザーインターフェース層**  
  UI や Web ダッシュボードは提供せず、アプリケーションやサービスへの組み込みを前提としたコードベースの利用を想定しています。

- **本番運用向け MLOps 基盤**  
  CI/CD、モデル監視、オンライン推論基盤などは、利用環境や要件に応じて外部で構成されることを前提としています。

- **データおよび成果物の永続化方式**  
  学習データ、モデル、評価結果の保存形式や保存先は、システム構成に応じて利用側で柔軟に設計できるよう、本エンジンでは規定しません。

- **分散・大規模計算環境への最適化**  
  単一マシンでの実行を主軸とし、分散学習やクラスタ環境への最適化は利用側で拡張できる設計を想定しています。

- **業務ドメイン固有の前処理・特徴量設計**  
  汎用性を重視し、特定業務に依存した前処理や特徴量エンジニアリングは利用側で実装することを想定しています。

- **業務判断の自動化**  
  本エンジンはモデル構築と意思決定支援を目的としており、最終的な業務判断は利用者が行う前提としています。

## 他ライブラリとの差別化ポイント
本エンジンは、**他ライブラリやアプリケーションに組み込まれることを前提とした AutoML コア**として設計されています。

* **AutoML コアに特化**
  前処理、学習、評価、最適化、説明といった中核機能を提供し、上位層での実装を前提とします。

* **制御可能で透明な AutoML**
  アルゴリズム、評価指標、探索戦略を明示的に指定でき、再現性を確保します。

* **組み込み利用を前提とした API**
  ライブラリとして他システムから呼び出される設計です。

* **タスク別 Engine 構成**
  各タスクの特性に応じた Engine を提供します。

* **説明と要約を標準機能として提供**
  特徴量寄与の算出と、結果の自然言語要約を支援します。

## 想定ユースケース
業務アプリケーションへの組み込みから PoC・分析基盤、学習結果の説明・共有、機械学習パイプライン設計の検証までをユースケースとして想定しています。

## クイックスタート
タスクごとに専用の Engine クラスを提供し、学習・推論・説明を一貫した API で扱えます。

```python
import automl_engine as ae

engine = ae.ClassificationEngine()

engine.fit(
    X=train_data,
    y=target,
)

predictions = engine.predict(test_data)
probabilities = engine.predict_proba(test_data)
explanations = engine.explain(test_data)
summary = engine.summarize()
```


```python
import automl_engine as ae

engine = ae.RegressionEngine()

engine.fit(
    X=train_data,
    y=target,
)

predictions = engine.predict(test_data)
explanations = engine.explain(test_data)
summary = engine.summarize()
```

※ 以上は最小構成の例です。前処理や評価指標、探索戦略は用途に応じて明示的に指定できます。
